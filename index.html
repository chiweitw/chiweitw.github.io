FastCulture

  • Team: 5-6 people
  • My Role: project owner/solution architect/full-stack engineer/project manger
  • FastCulture is a web APP for providing people a better way to discover interesting places in the city. Also a convenient tool for publisher to batch create place data and manage content.

Tech stack

  • React JS
  • Ruby on Rails
  • AWS EC2/RDS/SES/S3/Route53/ELB
  • PostgreSQL
  • Adobe XD/ Zeplin
  • Google oauth/Google place API/ Facebook oauth

Features

  • User can discover trending places in the city by categories and tags
  • User can search by typing relevant keywords into the search bar and get the best places
  • User can browse posts & guides created by local tastemakers
  • User can create posts and get them saved into the personalised recommendation list that user can refer to and easy to share
  • Publisher can import data from the spreadsheet
  • Publisher can populate the content in admin panel

City Gems

  • Team: 3 people
  • My Role: full-stack engineer
  • City gems is a wechat mini-program for users to share their city's hot spots to friends and explore the trendings.

Tech stack

  • Wechat mini-program
  • Ruby on Rails
  • PostgreSQL
  • Adobe XD
  • Trello

Features

  • User can get and share their hot spots.
  • User can see the trending places based on people sharing
  • User can CRUD their own collections with specified categories(eat, dring, play) and locations.
  • User can get route directly by opening with third-party map service.
  • User can filter information by categories: eat, dring, play and locations.
